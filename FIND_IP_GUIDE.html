Step 1: Find Your Mac's IP Address

  1. Click the Apple logo (top-left corner of your screen)
  2. Click System Preferences
  3. Click Network
  4. Select your active connection (Wi-Fi will have a green dot ‚óè)
  5. Look for IP Address

It will look something like:

192.168.1.5

or

10.0.0.10

Step 2: Update Your Code

Open this file in Android Studio:

app/src/main/java/com/example/smartmeal/network/ApiClient.kt

Find line 12:

const val BASE_URL = "http://localhost:8080/smartmeal/backend/api/"

Change it to (use YOUR IP):

const val BASE_URL = "http://192.168.1.5/smartmeal/backend/api/"

Replace 192.168.1.5 with YOUR Mac's IP from Step 1!

Then save the file (Cmd+S)

Step 3: Rebuild Your App

In Android Studio Terminal (or any terminal):

cd /Users/mac/StudioProjects/SmartMeal
./gradlew clean assembleDebug installDebug

Step 4: Test from Phone Browser

Before running the app, test the connection:

  1. Make sure your phone and Mac are on the SAME Wi-Fi
  2. Open Chrome or Safari on your phone
  3. Go to: http://YOUR_MAC_IP/smartmeal/backend/api/users.php
  4. Example: http://192.168.1.5/smartmeal/backend/api/users.php

You should see JSON text like:

{"message":"SmartMeal User API","status":"active"...}
‚úÖ Success! If you see JSON in your phone's browser, your app will work too!

Step 5: Run Your App!

Now test the complete flow:

  1. Splash ‚Üí Auto-navigates to Login
  2. Signup ‚Üí Create account ‚Üí Onboarding 1
  3. Onboarding ‚Üí 1 ‚Üí 2 ‚Üí 3 ‚Üí Home
  4. Login ‚Üí Goes directly to Home

Troubleshooting

If phone browser can't access the URL:

  • Check both devices are on same Wi-Fi
  • Check XAMPP Apache is running on your Mac
  • Try disabling Mac's firewall temporarily (System Preferences ‚Üí Security ‚Üí Firewall)
  • Test on Mac first: http://localhost/smartmeal/backend/api/users.php
